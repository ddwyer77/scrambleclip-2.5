# Input Audio Directory

Place your audio file in this directory. This audio will be used for all generated clips.

## Supported Formats
- MP3 (.mp3) - Recommended
- WAV (.wav)

## Notes
- The default expected filename is "audio.mp3"
- You can select a different file using the GUI
- If your audio is shorter than 16 seconds, it will be looped
- If your audio is longer than 16 seconds, it will be trimmed
- For best results, use audio that's approximately 16 seconds long
